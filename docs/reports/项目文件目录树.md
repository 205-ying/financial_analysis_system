## 🚀 关键目录说明

### 后端核心目录 (`backend/`)
- **`app/`**: 应用核心代码，遵循Clean Architecture分层设计
  - **`api/`**: API路由层，处理HTTP请求和响应
  - **`core/`**: 核心基础设施（配置、数据库、安全、异常）
  - **`models/`**: SQLAlchemy数据模型（ORM）
  - **`schemas/`**: Pydantic数据传输对象（DTO）
  - **`services/`**: 业务逻辑服务层
- **`alembic/`**: 数据库迁移管理
- **`scripts/`**: 维护工具和测试脚本
- **`tests/`**: 单元测试和集成测试

### 前端核心目录 (`frontend/src/`)
- **`api/`**: API接口层，封装HTTP请求
- **`components/`**: Vue组件库（图表、弹窗、通用组件）
- **`views/`**: 页面视图（仪表板、订单、费用、KPI等）
- **`stores/`**: Pinia状态管理（认证、权限、业务状态）
- **`router/`**: Vue路由配置和守卫
- **`types/`**: TypeScript类型定义
- **`utils/`**: 工具函数库

### 文档系统 (`docs/`)
- **核心文档**: 开发指南、架构文档、命名规范
- **历史存档**: 30+ 阶段性交付和测试报告
- **优化报告**: 项目结构优化和代码清理报告

## 🎯 项目特色

### 技术架构
- **后端**: FastAPI + SQLAlchemy 2.0 (异步) + PostgreSQL + JWT认证
- **前端**: Vue3 + TypeScript + Vite + Element Plus + ECharts
- **权限系统**: RBAC权限控制 + 细粒度权限管理
- **数据权限**: 门店级数据访问控制

### 核心功能模块
- **订单管理**: 订单录入、查询、统计
- **费用管理**: 费用科目、费用记录管理
- **KPI分析**: 多维度业绩指标计算和展示
- **报表中心**: 日汇总、月汇总、门店绩效报表
- **数据导入**: Excel/CSV批量导入，支持错误报告
- **审计日志**: 完整的操作审计追踪
- **权限管理**: 用户角色权限、数据权限管理

### 开发规范
- **分层架构**: API → Service → Model 严格分层
- **异步编程**: 全面使用SQLAlchemy 2.0异步API
- **类型安全**: 前后端TypeScript类型定义
- **测试覆盖**: 单元测试 + 集成测试 + E2E测试
- **代码质量**: Ruff代码检查 + Prettier格式化
